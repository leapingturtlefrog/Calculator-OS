# Calculator OS v2.0

A minimal OS that boots to a calculator with bonus features.

## Commands
- Math: `5+3`, `10-2`, `6*7`, `8/2`
- `iching` - Get an I Ching fortune
- `moji` - Random asciimoji
- `lasagna` - ASCII art lasagna

## Build & Run
```
make run
```

Requires: gcc (32-bit), nasm, qemu-system-i386
